# YGO Proxy Printer
v0.1, build 2025_07_29

## Installation
Create environment:
```powershell
conda create -n QtCreator python=3.8.8 pip
```

Activate environment:
```powershell
conda activate QtCreator
```

Install requirements:
```powershell
pip install -r requirements.txt
```

## Run User Interface
```powershell
python YGOProxyPrinter_UI.py
```
